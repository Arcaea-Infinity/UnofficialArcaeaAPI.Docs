## assets/icon

> Note: It is not recommended to use this API frequently.

| arguments | description                  | optional |
|:----------|:-----------------------------|----------|
| partner   | partner id                   | false    |
| awakened  | Boolean, partner is awakened | true     |

###### Tag

* Enable Cors

#### Example

+ `{apiurl}/botarcapi/assets/icon?partner=1`

###### Content-Type

```
image/png
```
